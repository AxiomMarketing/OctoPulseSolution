# Post Planning Hebdo

## Quand l'utiliser
- **Lundi 10h** (après plan-hebdo Stratege 9h)
- Ajustement intra-semaine si event Radar ou pivot Stratege

## Entrées requises
- Calendrier éditorial mensuel (`maeva:calendrier-editorial`)
- Plan tests paid Stratege semaine en cours (pour éviter contradictions de message)
- Insights content Radar vendredi (`insights-content/YYYY-WXX.md`)

## Étapes

1. **Slots par plateforme** (cadence stable):
   - Instagram : 5-7 posts/sem (mix reel/carousel/story)
   - TikTok : 3-5 reels/sem
   - Pinterest : daily (via pin-master Funnel)
   - Blog : 1 article/sem ou bimensuel
   - Stories : daily

2. **Mix formats** hebdo :
   - 2-3 reels
   - 2 carousels
   - 1 post single
   - stories quotidiennes
   - 1 UGC / témoignage / behind-the-scenes

3. **Coordination paid** :
   - Vérifier qu'aucun post organique ne contredit un test paid actif
   - Si angle testé en paid performe (remontée Stratege) → booster version organique
   - Ne jamais annoncer une promo avant activation paid

4. **A/B testing organique** : varier hook / visuel / heure de publication
   - Tracker via saves + shares (pas likes)

5. **Heures de publication** optimales (à affiner via learnings) :
   - Instagram : 7h30 / 12h30 / 19h
   - TikTok : 18h-21h
   - Pinterest : mâtin week-end

## Sortie
- `team-workspace/marketing/maeva/planning/YYYY-WXX.md`
- Buffer / Planoly / Later si automatisé
- CC Sparky + Stratege (visibilité cross-canal)

## Règles strictes
- **Cadence stable** : mieux vaut 5 posts/sem réguliers que 10 irréguliers
- **Qualité > volume**
- **Pas de post automatisé généré par IA sans relecture** (anti-drift de marque)
- **Slots réservés events Radar** : garder 1-2 slots flexibles chaque semaine
